## About

**TripLingo** is an interactive language-learning web app designed for travellers who want to pick up a new language before or during their trip. Users register, choose a language and traveller profile, then work through levelled learning activities at their own pace.

## Features

| Feature | Description |
|---|---|
| **User Accounts** | Register and log in with a personal account. Session is persisted via localStorage. |
| **Language & Traveller Selection** | Pick a language and a traveller type during onboarding to tailor content. |
| **Levels** | Progress through structured difficulty levels. |
| **Words & Phrases** | Browse essential travel vocabulary grouped by topic. |
| **Reading Comprehension** | Read passages and answer questions to build understanding. |
| **Listening Comprehension** | Audio-based exercises to train your ear. |
| **Memory Game** | Card-matching game to reinforce vocabulary. |
| **Vocabulary Quiz** | Test your knowledge with multiple-choice questions. |
| **Notepad** | Take in-app notes while you study. |
| **Saved Words** | Save words and phrases to review later. |
| **Dark Mode** | Toggle between light and dark themes. |
| **User Dashboard** | View your profile and track your learning journey. |

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Tech Stack

- **[Vue 3](https://vuejs.org/)** — Component-based frontend framework
- **[Vite](https://vitejs.dev/)** — Fast build tooling
- **[Pinia](https://pinia.vuejs.org/)** — State management
- **[Vue Router](https://router.vuejs.org/)** — Client-side routing
- **[Bootstrap 5](https://getbootstrap.com/) + [BootstrapVueNext](https://bootstrap-vue-next.github.io/bootstrap-vue-next/)** — UI components and styling
